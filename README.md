# Wo0lfgang Public Docs

各产品的公开文档（隐私政策、用户协议、支持页），通过 GitHub Pages 托管。

**站点根 URL：** https://wo0lfgang.github.io/docs/

## 产品目录

| 产品 | 目录 | 隐私政策 | 支持页 |
|------|------|----------|--------|
| 金鹅 / Gold Goose | `goldgoose/` | [中文](https://wo0lfgang.github.io/docs/goldgoose/privacy-policy-zh.html) | [Support](https://wo0lfgang.github.io/docs/goldgoose/support.html) |

## 部署

1. 本仓库为 **Public**
2. GitHub → **Settings → Pages** → Source: `main` 分支，`/ (root)` 文件夹
3. 推送后约 1～5 分钟生效

## 新增产品

在根目录新建 `<product-slug>/`，放入 `privacy-policy-*.html`、`terms-of-service-*.html`、`support.html`，并更新本 README 与根 `index.html`。
